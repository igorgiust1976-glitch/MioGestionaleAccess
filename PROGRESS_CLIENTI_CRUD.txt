================================================================================
  WORKFLOW: CLIENT CRUD OPERATIONS
================================================================================

1. USER CLICKS "ANAGRAFICHE" BUTTON
   → Form_Anagrafiche_Clienti opens as modal
   → CaricaClienti() executes: SELECT * FROM Cliente ORDER BY Rag_Soc
   → DataGridView displays all clients with single column: Rag_Soc visible

2. USER CLICKS "AGGIUNGI"
   → Form_ClienteDettagli(null, connString) opens
   → Text = "Nuovo Cliente"
   → PulisciCampi() clears all TextBox
   → User enters: Rag_Soc (required), Indirizzo, Referente, Tel_1, Tel_2, Note_1, Note_2
   → Salva button triggers: InserisciCliente()
     - INSERT INTO Cliente (Rag_Soc, Indirizzo, Referente, Tel_1, Tel_2, Note_1, Note_2)
     - VALUES (?, ?, ?, ?, ?, ?, ?)
   → MessageBox: "Cliente aggiunto con successo"
   → DialogResult.OK returned to Form_Anagrafiche_Clienti
   → CaricaClienti() called again to refresh list

3. USER SELECTS CLIENT AND CLICKS "MODIFICA"
   → Form_ClienteDettagli(clienteId, connString) opens with client ID
   → Text = "Modifica Cliente"
   → CaricaDatiCliente(clienteId) reads existing record
   → TextBox populated with: Rag_Soc, Indirizzo, Referente, Tel_1, Tel_2, Note_1, Note_2
   → User edits fields
   → Salva button triggers: ModificaCliente()
     - UPDATE Cliente SET Rag_Soc=?, Indirizzo=?, Referente=?, Tel_1=?, Tel_2=?, Note_1=?, Note_2=?
     - WHERE ID=?
   → MessageBox: "Cliente modificato con successo"
   → DialogResult.OK returned to Form_Anagrafiche_Clienti
   → CaricaClienti() called again to refresh list

4. USER SELECTS CLIENT AND CLICKS "ELIMINA"
   → MessageBox confirmation: "Sei sicuro?"
   → If OK:
     - DELETE FROM Cliente WHERE ID = clienteId
     - CaricaClienti() called to refresh list
   → If Cancel: operation aborted

5. USER CLICKS "CHIUDI"
   → Form_Anagrafiche_Clienti closes
   → Returns to Form1
